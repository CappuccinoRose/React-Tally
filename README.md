# React-Tally - 记账应用

一个现代化的React记账应用，帮助用户轻松管理个人财务，追踪收入支出，分析消费趋势。

## 功能特性

### 📊 财务概览
- 年度账单统计（收入、支出、结余）
- 月度收支趋势图表
- 支出排行榜TOP5

### 📱 响应式设计
- 适配各种屏幕尺寸
- 小屏幕下图表可横向滚动
- 桌面屏幕全宽显示

### 🎨 现代化界面
- 黑金配色方案
- 清晰的数据可视化
- 简洁直观的用户界面

## 技术栈

- **前端框架**: React 18
- **样式**: CSS Modules / CSS变量
- **构建工具**: Vite
- **图标**: React Icons
- **响应式**: Flexbox + CSS Grid

## 安装指南

1. 克隆仓库到本地
   ```bash
   git clone https://github.com/CappuccinoRose/React-Tally.git
   ```

2. 进入项目目录
   ```bash
   cd react-tally
   ```

3. 安装依赖
   ```bash
   npm install
   ```

4. 启动开发服务器
   ```bash
   npm run dev
   ```

5. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)
